Merge heaps at intervals [i, j[ and [j, k[ in array a into a new heap at interval [i, k[.

Hypothesis :

  • i <= j <= k
  • j - 1 is the last leaf ot the first heap

Parameters:

  • arity must be an int.
    (arity of the heap)

  • compare must be a function.
    (the comparison function)

  • swap must be a function.
    (the swap function)

  • a must be an array.
    (the array where the two heaps are stored)

  • i must be an int.
    (is the root of the first heap)

  • j must be an int.
    (is the root of the second heap)

  • k must be an int.
    (- 1 is the index of the last leaf in the second heap)

daryheap.merge = function ( arity, compare, swap, a, i, j, k ) { for ( ; j < k ; ++j ) { daryheap.percolateup( arity, compare, swap, a, i, j + 1, j ); } };
